Badge

<p class=“tiadian-text-lead”>Easily create nice looking notification badges.</p>

Usage

To create a badge, just add the .tiadian-badge class to an inline element like <a> or <span> element.

<span class="tiadian-badge"></span>
1 100 
<span class="tiadian-badge">1</span>
