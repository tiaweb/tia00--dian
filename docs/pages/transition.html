Transition

Create smooth transitions between two states when hovering an element.

Usage

To toggle a transition on hover or focus, add the .tiadian-transition-toggle class to a parent element. Also add tabindex="0" to make the animation focusable through keyboard navigation and on touch devices. Add one of the .tiadian-transition-* classes to any child element to apply the actual effect.

Class Description
.tiadian-transition-fade Lets the element fade in
.tiadian-transition-scale-up
.tiadian-transition-scale-down		 The element scales up or down.
.tiadian-transition-slide-top
.tiadian-transition-slide-bottom
.tiadian-transition-slide-left
.tiadian-transition-slide-right		  Lets the element slide in from the top
.tiadian-transition-slide-top-small
.tiadian-transition-slide-bottom-small
.tiadian-transition-slide-left-small
.tiadian-transition-slide-right-small		  The element slides in from the top, bottom, left or right with a smaller distance.
.tiadian-transition-slide-top-medium
.tiadian-transition-slide-bottom-medium
.tiadian-transition-slide-left-medium
.tiadian-transition-slide-right-medium		  The element slides in from the top, bottom, left or right with a medium distance.

This component is mostly used in combination with the Overlay component as elements are transitioned from invisible to visible state. To place the overlay on top of another element, like an image, use the Position component.

<div class="tiadian-transition-toggle" tabindex="0">
    <div class="tiadian-transition-fade"></div>
</div>
